# Boston Rent Analysis: Impact of Crime, Education, & Walkability

A comprehensive data analytics project investigating the factors influencing rent prices across New York City's five boroughs, with focus on crime rates, education quality, and walkability metrics.

## 🌎 Project Overview

This analysis examines rental market dynamis in Boston (2019-2024), exploring how crime, education quality, and walkability influence rent prices across different neighborhoods. The project provides data-driven insights for real estate investors, policy makers, and the general public.
